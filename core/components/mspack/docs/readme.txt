A basic Extra for MODx Revolution + miniShop2.

Feel free to suggest ideas/improvements/bugs on GitHub:
http://github.com/Marabar/msPack/issues

--------------------

После утановки msPack в настройках miniShop2 появится дополнительная вкладка Упаковка,
где необходимо указать виды упаковок и их индекс для отображения на сайте в этой последовательности.

Добавить дополнительную колонку pack_cost в таблице ms2_orders, выполнив запрос
phpMyAdmin:

ALTER TABLE `modx_ms2_orders` ADD `pack_cost` DECIMAL(12,2) NULL DEFAULT '0.00' AFTER `delivery_cost`;

Сниппеты и Параметры
[[msPack]]

Сниппет вызывать не кешированным

&tpl - Имя чанка для одного пункта, по умолчанию: msPackTpl
&sortby - Поле для сортировки, по умолчанию: packindex
&sortdir - Направление сортировки, по умолчанию: ASC


Доступные плейсхолдеры
msPack:

[[+id]] - ID упаковки
[[+name]] - Название упаковки
[[+description]] - Описание
[[+pack_price]] - Цена
[[+packindex]] - Индекс, по этому полю происходит сортировка
[[+active]] - Статус одной упаковки вкл/откл

В любом месте страницы, где вызывается сниппет доступен
[[+pc.packprice]] - Цена выбранной упаковки
[[+pc.check]] - Используется только при первой загрузки корзины, при условии что
                ранее упаковка не была выбрана в рамках текущего заказа.

В чанке письма доступны:
[[+pack_cost]] - Цена упаковки к оплате
